# Latha Hotel Website

Premium static restaurant website built with HTML, CSS and JavaScript.

## Files

- `index.html` — website structure and content
- `styles.css` — responsive luxury styling and animations
- `script.js` — menu filtering, mobile navigation, scroll effects and parallax

## Preview locally

Open `index.html` in a browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Free hosting with GitHub Pages

1. Upload all files to the root of your GitHub repository.
2. Open **Settings → Pages**.
3. Choose **Deploy from a branch**.
4. Select `main` and `/ (root)`.
5. Save.

## Before publishing

Replace the placeholder new-branch details:

- Exact address
- Phone and WhatsApp number
- Opening hours
- Confirmed menu and prices
- Restaurant-owned images
- Social media links

The current menu intentionally avoids pairing individual dishes with uncertain photos.
